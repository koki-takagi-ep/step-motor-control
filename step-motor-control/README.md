# モーター制御プログラム

28BYJ-48ステッピングモーターの精密角度制御を実現するArduinoプログラムです。

## 📂 バージョン構成

### v1 - シリアル通信版
- **対応ボード**: すべてのArduino（UNO R3、R4、Nano等）
- **通信方式**: USB経由のシリアル通信
- **用途**: 有線接続でのモーター制御、開発・テスト用

### v2 - Bluetooth LE対応版
- **対応ボード**: Arduino UNO R4 WiFi（BLE内蔵）
- **通信方式**: シリアル通信 + Bluetooth LE
- **用途**: ワイヤレス制御、スマートフォン連携、IoTプロジェクト
- **追加機能**: Python GUI/CLIツール付属

## 🎯 選び方

| こんな時は... | おすすめ |
|-------------|---------|
| 初めて使う | v1 |
| Arduino UNO R3を使用 | v1 |
| ワイヤレスで制御したい | v2 |
| スマホから操作したい | v2 |
| PCに常時接続OK | v1 |
| GUIで操作したい | v2 |

## 🚀 クイックスタート

### v1を使う場合
```bash
1. Arduino IDEでmotор/v1/main.inoを開く
2. AccelStepperライブラリをインストール
3. 書き込み → シリアルモニターで制御
```

### v2を使う場合
```bash
1. Arduino IDEでmotor/v2/main.inoを開く
2. AccelStepper + ArduinoBLEライブラリをインストール
3. 書き込み → BLEアプリまたはPythonツールで制御
```

## 📋 共通仕様

### ハードウェア
- **モーター**: 28BYJ-48（5V）
- **ドライバー**: ULN2003
- **制御ピン**: 8, 9, 10, 11
- **1回転**: 4096ステップ

### 基本機能
- 絶対・相対角度制御
- 最短経路計算
- 電源節約モード
- 5段階速度設定
- リアルタイムステータス

詳細は各バージョンのREADMEを参照してください。
