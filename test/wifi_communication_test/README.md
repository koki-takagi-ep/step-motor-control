# Arduino WiFi Communication Test

このプロジェクトでは、ESP8266/ESP32ボードとPCとのWiFi通信テストを実装しています。二つの通信方法が用意されています：

1. Webサーバーインターフェース: ブラウザから直接ボードにアクセスして操作
2. TCP双方向通信: PCのPythonサーバーとボード間で双方向の通信

## 必要なもの

- ESP8266または ESP32ボード
- マイクロUSBケーブル
- Wi-Fi環境
- ArduinoIDE（最新バージョン推奨）
- Python 3.6以上（PCサーバー用）

## セットアップ手順

1. Arduino IDEにESP8266/ESP32ボードをインストール
2. スケッチファイル内のWiFi設定を変更
3. PCサーバープログラムを実行して通信テスト

詳細な手順は同梱のドキュメントを参照してください。
